# Dege-Ganjiur-Tibetan-Ancient-Book-Dataset-Reconstruction

## 1. 数据集简介
本数据集包含593对德格版《甘珠尔》藏文古籍的：
- 原始退化扫描图像
- 算法重建后的干净真值图像
- 对应校对完成的文本文件（txt）
- 藏文古籍重建核心算法脚本

适用于：文档图像复原、古籍数字化、藏文OCR、深度学习训练。

## 2. 文件结构
├── dataset/          # 古籍图像与文本数据集
│   ├── degraded/     # 原始退化图像
│   ├── clean/        # 重建干净图像
│   └── txt/          # 校对后藏文文本
├── code/             # 核心重建算法脚本
│   ├── boundary_extract.py    # 非线性边界提取
│   ├── grid_generate.py       # 自适应非矩形网格生成
│   └── render_engine.py       # 藏文字丁语义排版引擎
└── README.md

## 3. 算法功能
1. 纸张非线性物理边缘提取
2. 边界吸附式非均匀网格生成
3. 藏文字丁语义块解析
4. 多级排版与保真渲染
5. 批量生成“退化-干净”成对训练数据

## 4. 使用方式
直接运行 code 下三个脚本即可重建古籍图像。
