# Dege-Ganjiur-Tibetan-Ancient-Book-Dataset-Reconstruction

## 1. 数据集简介
本数据集包含593对德格版《甘珠尔》藏文古籍的：
- 原始退化扫描图像
- 算法重建后的干净真值图像
- 对应校对完成的文本文件（txt）
- 藏文古籍重建核心算法脚本

适用于：文档图像复原、古籍数字化、藏文OCR、深度学习训练。

## 2. 文件结构
├── dataset/          # 古籍图像与文本数据集
│   ├── degraded/     # 原始退化图像
│   ├── clean/        # 重建干净图像
│   └── txt/          # 校对后藏文文本
├── code/             # 核心重建算法脚本
│ ├── background.py # 古籍背景合成与生成模块
│ ├── boundary_grid.py # 非线性边界提取 + 自适应网格生成模块
│ ├── main.py # 核心流程调度与主入口模块
│ └── tibetan_character.py # 藏文字丁语义解析与排版引擎模块
└── README.md

## 3. 算法功能
1.  纸张非线性物理边缘提取与自适应非矩形网格生成
2.  藏文字丁语义块解析与多级排版渲染
3.  古籍背景合成与真实感增强
4.  批量生成「退化-干净」成对训练数据
5.  藏文古籍非线性几何重建全流程调度

## 4. 使用方式
本仓库提供的是**核心算法模块脚本**，而非一键运行的完整程序。
使用时需按以下流程调用：
1.  运行 `boundary_grid.py` 提取古籍边界并生成自适应网格
2.  调用 `tibetan_character.py` 解析藏文字丁并完成语义排版
3.  通过 `background.py` 合成真实古籍背景
4.  以 `main.py` 为入口，串联全流程完成古籍图像重建
